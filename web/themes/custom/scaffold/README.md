## Theme build
 - Run `yarn install` this will download and install all the code required to build the theme.
 - Run `gulp` to watch the theme folder for CSS/JS changes while building the theme.

### Select / Tom selct build
 - Run 'npm run build' in /libraries/tom-select folder.

## Breakpoints
These settings can be changed in 'variables/_breakpoints.scss'

| Name          | Size       |
|---------------|------------|
| $phone        | \>= 480px  |
| $tablet       | \>= 768px  |
| $large_tablet | \>= 843px  |
| $laptop       | \>= 1280px |
| $desktop      | \>= 1440px |
